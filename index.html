How to Use with Minecraft

üìÅ File Location

Paintings are imported from and exported to a "paintings" folder inside your Minecraft directory:

  • Windows: %appdata%\.minecraft\paintings\
  • Mac: ~/Library/Application Support/minecraft/paintings/
  • Linux: ~/.minecraft/paintings/

üéÆ In-Game Commands

Export: /paintexport <filename>

While holding a painted canvas, exports it to the paintings folder

Import: /paintimport <filename>

In survival: Hold an empty canvas and full palette. In creative: Just run the command

üé® Workflow Examples

Import External Art to Minecraft:

  1. Use this web tool to convert your image to .paint format
  2. Download the .paint file and place it in your paintings/ folder
  3. In Minecraft, hold empty canvas + full palette (survival) or just use creative
  4. Run /paintimport filename (without .paint extension)
  5. Your canvas now has the imported artwork!

Share Your Minecraft Art:

  1. Paint something awesome in Minecraft
  2. Hold the painted canvas and run /paintexport myart
  3. Find myart.paint in your paintings folder
  4. Use this web tool to convert it to PNG/JPG for sharing

Create Multi-Canvas Murals:

  1. Upload a large image to this web tool
  2. Use Multi-Canvas feature to split it into a grid
  3. Download all the generated .paint files
  4. Place them in your paintings folder
  5. Import each piece and arrange them on your wall in order
  6. Enjoy your high-resolution mural!

üí° Tips

  • Canvas Types: Choose the right canvas size for your image aspect ratio
  • Editable Paintings: Set Generation to 0 to allow in-game editing
  • Locked Paintings: Set Generation to any number above 0 to prevent editing
  • File Names: Use simple names without spaces for easier importing
  • Multi-Canvas: Perfect for pixel art that needs more detail than 32x32
